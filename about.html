About This Website

This website is a simple student management system created as part of a web development project. It allows users to view student information and manage class schedules in an organized and user-friendly way.

The website is built using HTML5, CSS3, and JavaScript, with Firebase used for data storage and basic authentication.

About Me

I am a student learning web development. This project helped me understand how to structure webpages, apply styling using CSS, and add interactivity using JavaScript.
